# Contributing

Thanks for contributing to HWP MCP Server.

## Development

```bash
npm install
npm run build
npm test
npm run lint
```

## Code Style

- TypeScript strict mode is required.
- Keep functions small and focused.
- Avoid logging sensitive document content.

## Testing

- Add unit tests for new logic.
- Add integration tests for file parsing or conversion.

## Pull Requests

- Describe the change and its impact.
- Include tests that cover the change.
